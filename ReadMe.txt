MoneyMaker - cкрипт для профессиональных трейдеров, каперов и игроков в покер.

Позволяет принимать от инвесторов денежные средства, в доверительное управление.

-------------------------------------------------------------------------------------

1. Залить /lib/full_db.sql в MySQL

2. В /lib/config.php вприсать параметры подключения к MySQL

3. Готово!
